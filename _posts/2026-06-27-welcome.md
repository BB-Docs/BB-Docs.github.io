This is a sample lesson. Every lesson is a single Markdown file in the `_posts/`
folder, named `YYYY-MM-DD-short-title.md`. The date in the filename sets where
it appears in the list — newest first.

## How to add a lesson

1. Copy this file in `_posts/`.
2. Rename it with today's date, e.g. `2026-06-28-discipline.md`.
3. Edit the bits at the top (between the `---` lines): `title`, `subtitle`, `date`.
4. Write the lesson below in plain Markdown.
5. Commit — the site rebuilds automatically.

## What you can write

You get headings, **bold**, *italics*, and lists:

- A point to remember
- Another one
- A third

> Pull-quotes look like this — good for the one line you want to land.

You can drop in an image by putting it in `assets/` and linking it:

`![caption](/assets/my-image.png)`

That's the whole system. One file per day.
